# WhatsApp Automation using Make, Airtable, and Twilio

This project demonstrates a simple automation workflow that sends WhatsApp messages or images to multiple contacts automatically.

The automation is built using Make (formerly Integromat), Airtable, and Twilio's WhatsApp API. Contact numbers are stored in Airtable, and the workflow retrieves those numbers, loops through them, and sends a WhatsApp message to each contact.

## 🚀 Features

- Send WhatsApp messages automatically
- Send images along with messages
- Fetch contacts dynamically from Airtable
- Loop through multiple phone numbers
- Fully automated workflow using Make

## 🛠️ Tools & Technologies

- Make (Automation Platform)
- Airtable (Cloud Database)
- Twilio (WhatsApp Messaging API)

## ⚙️ Workflow

1. Manual trigger starts the automation
2. Message and image fields are defined
3. Contacts are retrieved from Airtable
4. Loop module processes each phone number
5. Twilio sends the WhatsApp message to every contact

## 📌 Use Cases

- Marketing message broadcasts
- Notifications to multiple users
- Automated reminders
- Customer communication
